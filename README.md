# 🛒 Retail Sales Analysis

## 📌 Descripción del proyecto
Análisis exploratorio de datos de ventas de una empresa retail con presencia en Colombia, Perú y Ecuador, con el objetivo de evaluar su desempeño durante el primer año de expansión.

## 🎯 Objetivo de negocio
Evaluar el rendimiento de las sucursales en los tres paises e identificar oportunidades de crecimiento.

## 🛠️ Herramientas utilizadas
- SQL (extracción, transformación y validación de datos)
- Power BI (visualización)

## 🔍 Proceso de análisis
1. Integración de múltiples fuentes de datos
2. Limpieza y validación de información
3. Creación de métricas clave (KPIs)
4. Desarrollo de dashboard interactivo

## 📊 KPIs analizados
- Ventas totales
- Ingresos por país
- Ticket promedio
- Productos más vendidos
- Marcas más populares
